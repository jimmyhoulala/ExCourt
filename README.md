# 易场地 - 同济大学体育馆场地交换系统

## 项目简介

**易场地** 是一款专为 **同济大学师生** 设计的场地管理与预约系统。系统旨在优化校内体育场馆资源的利用率，提供便捷的场地预约、交换、拼场功能，并实现实时数据更新与高效管理。本仓库中的代码为2024年秋季软件工程项目的成果，仅为一个初始版本，功能并不完善，后续将在未来进行继续的更新迭代。

---

## 项目背景

- **预约场地现状**：现有系统中羽毛球预约需求激烈，但存在抢场难、资源浪费、信息不透明等问题。
- **用户需求**：
  - 96% 的用户希望拥有一个能实现 **场地交换** 和 **组队** 的平台。
  - 75% 的用户经常需要更换场地或组队。

系统通过解决这些痛点，为校内用户提供灵活、高效的场地管理方案。

---

## 功能介绍

1. **场地信息可视化**
   - 以图形化方式展示场地状态，支持筛选和加入空缺场地。
2. **场地资源再分配（换场）**
   - 用户可发布换场需求，系统自动匹配并确认。
3. **组队功能**
   - 用户可发布拼场需求，自动更新场地状态并接受加入申请。
4. **场地资源转让**
   - 用户可放弃场地并将其转让给其他用户。
5. **失物招领**
   - 发布、查看失物信息，实现物品核销和申诉。
6. **即时通讯**
   - 用户间实时交流，便于协调场地与拼场事宜。
7. **账户管理**
   - 支持用户的个人信息与信誉分管理，保障交易顺畅。

## 目标用户

- **普通用户**：同济大学在读学生和老师。
- **系统管理员**：负责资源管理和系统维护。
